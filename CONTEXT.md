## Projectdoelen

Dit project levert een volledig draaiboek voor het diner op:

1. **Menu** — visueel aantrekkelijk, geschikt voor ~24 personen
2. **Recepten** — uitgewerkt per gerecht
3. **Boodschappenlijst** — per categorie, op hoeveelheid voor 24 pers.
4. **Kookplanning** — tijdlijn van voorbereiding t/m serveren
5. **Logistiek** — tafeldekking, dranken, benodigdheden

## Randvoorwaarden

- Geen ananas in enig gerecht
- 1 vegan persoon — apart gerecht of volledig vegan menu meewegen
- Visuele presentatie telt — dit is een feestelijk familiediner
- Strak tijdschema: diner op tafel om 17:00
- Hosts moeten daarna vroeg op kunnen → weinig nazorg / opruimtijd
